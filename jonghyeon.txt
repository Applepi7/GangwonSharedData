1 Resiter Activity >

1. 성별
2. 나이
3. 여행 목적 ( 가족 여행, 관광, 낭만, 도심, 럭셔리, 레져, 비즈니스, 식도락 )
4. 예산 ( 저, 중, 고 선택 )
5. 직업 ( 직접 입력 )
6. 이름

순서 상관없이 이쁘게 배치 ㄱ.

2 Main Activity >

맨 위에 추천 카드 보여주자. 1Activity에서 얻은 빅데이터 바탕으로 추천함.
그리고 직접 검색하는것도 있고.
그리고 그 밑에 분류하는거 만들어서 위에 있는 여행 목적, 예산, 현재 위치와 가까운 거리, 별점, 위치(강원도 구역들)에 따라 볼 수 있음.

카드에서 보여줄 내용 : 사진, 현재 위치와의 거리, 전화번호, 업종 ( 음식, 관광명소, 숙박업소 ) , 
업종 세부 분류 ( 한식, 중식, 펜션, 호텔 등 ), 위치 알림 등록 ( 핀모양으로 하던지, 누르면 핀 모양에 색깔 들어옴 )
너무 많으면 알아서 빼셈


3 Card Activity >
카드 세부내용 페이지.
이름, 현재 위치로부터 떨어진 거리, 전화번호, 업종, 세부분류 가 보여줌.
그리고 밑에 지도뷰로 현재 위치랑 핀으로 나타내줌
그 밑에는 별점을 주고 볼수 있고, 바텀바로 전화할 수 있음.
별점 밑에는 더 많은 반응 보기 버튼를 통해 해당하는 업소에 대한 반응(댓글)을 별점과 함께 보여주는
액티비티(3-2)로 이동함.
별점 주기를 누르면, 별점을 주는 액티비티(3-1)로 넘어감.

3-1 Star Activity >
별점을 주고 리플을 입력할 수 있음.

3-2 RepleActivity >
리스트뷰로 다른 사람이 단 댓글을 보여줌. 다른 사람의 이름, 댓글, 나이, 성별, 별점 을 보여줌.

4 NavigationDrawer Activity >

맨 위에 이름, 성별, 나이, 직업이 이쁘게 뜸.
그 밑에 리스트 버튼이 이렇게 있음.

1. 최근 등록한 스팟 -> 4-1
2. 알림 뜬 스팟 -> 4-2
3. 마이 페이지  -> 4-3

역시 순서는 맘대로.

4-1 최근 등록 스팟 Activity >
최근 위치 알림을 등록한 스팟이 카드뷰로 뜸. 누르면 역시 3Activity로 이동.

4-2 알림 뜬 스팟 Activity >
최근 알림이 뜬 스팟이 카드 뷰로 뜸.

4-3 마이 페이지 Activity >

Resister Activity 가 나오며 바텀 뷰로 등록하기가 나옴.
정보 재수정 가능.

5 알림 Activity >
위치 알림 등록 스팟이 근처에 있을 시 알림을 주며, 알림 안에 버튼 ( 자세한 정보 확인하기 ) 를 누를 시,
해당하는 스팟의 카드 액티비티로 나옴.
